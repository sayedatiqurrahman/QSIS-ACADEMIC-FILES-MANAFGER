Create GitHub Account

Go to github.com/signup and create a free account with your Google account.

2

Fork This Repository

Click the button below to fork (copy) this repo to your own GitHub account.

Fork Repository
3

Upload Your Files

In your forked repo, navigate to the correct folder:

upload_academic_files/
  {semester}/
    sheet/{course-code}/ — Study sheets
    Previous Question ({semester})/2026/Mid/
    NOTES/ — Lecture notes
    Syllabus/ — Course syllabi
4

Create Pull Request

After uploading, click Contribute > Open pull request to submit.

View Existing PRs
5

Get Credit!

Type your Name, QSIS ID & WhatsApp in the PR description. Your name will appear in the Contributors section.

How to Contribute

Share your notes, sheets & previous questions with fellow students.

1

Create GitHub Account

Go to github.com and sign up — it's free.

2

Fork This Repo

Click Fork to get your own copy.

3

Upload Your Files

Go to upload_academic_files/{semester}/ and click Add file > Upload files.

4

Submit Pull Request

Click Contribute > Open pull request. Enter your Name, QSIS ID & WhatsApp.
